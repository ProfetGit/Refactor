# Changelog

All notable changes to Refactor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-12-10

### Added
- **Auto-Sell Junk**: Automatically sell grey items with options for low iLvl gear and collected transmogs
- **Auto-Repair**: Automatic gear repairs with guild bank support
- **Auto-Quest**: Quest automation with accept, turn-in, gossip skipping, and modifier bypass
- **Fast Loot**: Instant looting in a single frame
- **Loot Toasts**: Elegant visual loot notifications with quality filtering
- **Skip Cinematics**: Remember and skip previously watched cinematics
- **Auto-Confirm**: Auto-accept ready checks, summons, role checks, resurrections
- **Auto-Invite**: Accept group invites from friends, Battle.net friends, and guildmates
- **Auto-Release**: Context-aware spirit release for PvP/PvE/Open World
- **Quest Nameplates**: Kill and loot objective icons on enemy nameplates
- **Tooltip Plus**: Enhanced tooltips with class colors, rarity borders, transmog status, and customization
- **Action Cam**: Dynamic camera presets (Basic/Full modes)
- **Combat Fade**: Fade action bars and player frame out of combat
- **Speed Display**: Real-time movement speed percentage near player frame
- **Minimap Button**: Quick access to settings
- **Comprehensive Settings UI**: Sidebar navigation with categorized options
- **Localization Support**: English language with framework for future translations
